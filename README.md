1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/zacks08/isaac-portfolio.git
   cd isaac-portfolio
   \`\`\`

2. **Install dependencies**
   \`\`\`bash
   npm install
   # or
   yarn install
   \`\`\`

3. **Start development server**
   \`\`\`bash
   npm run dev
   # or
   yarn dev
   \`\`\`

   http://localhost:3000
   \`\`\`

\`\`\`bash
